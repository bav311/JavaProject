Реализуй метод main(String[]), который делит
массив array на два подмассива и заполняет ими двумерный массив result.
Если длина массива нечетная, то большую часть нужно скопировать
в первый подмассив.
Для разделения массива используй метод Arrays.copyOfRange(int[], int, int).
Порядок элементов не меняй.
При тестировании значения полей класса Solution будут разными, учти это.
